vlc-meta-reader
===============

Simple demonstration on how to read metadata from A/V files using libvlc.

Build instructions
-------------------

```
git clone https://github.com/xhochy/vlc-meta-reader.git
cd vlc-meta-reader
mkdir build
cd build
cmake .. && make
```

Dependencies
------------

* A working C++ compiler
* VLC 2.0 or newer

* Ubuntu packages: `build-essential cmake libvlc-dev`
